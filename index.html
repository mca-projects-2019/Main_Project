Third Eye

A helping aid for the visually impaired.

Features

Text-to-Speech

Text-to-Speech is a module, which converts the normal language text into audio.

Speech-to-Text

Speech-to-Text is a module, which converts the given audio into text format.

Translation

Translation is a module, which converts the given English text into Malayalam text.

Digital Library

A storage space where the converted audio are recoreded for future needs.

About Us

Third Eye, a technology aided system, which functions on voice commands specially designed to help the visually impaired students. This system will converted the text books(image or text format)into audio files using image to speech conversion and text to speech conversion techniques. It also converts the text books in English language to audio in Malayalam language, andsets upa didital library to store these converted audio files, which will help the students in their studies. This reduces the human effort and also proves to be quite handy in case of change in syllabi. This system also helps the students to write their examination without the help of a scribe, by automatically recording the dictated answer and automatically convertig it into text format.
The system will mark a beginning of new hopes and an emancipation from the above said requisities.
